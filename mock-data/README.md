# Mock Phishing Datasets

This directory contains evaluation datasets for testing the email-scoring agent:

1. `sample-emails.json`: General mock data including safe, spam, and suspicious emails.
2. `real-phishing-samples.json`: A dataset patterned after known real-world phishing categories (e.g. CEO fraud, delivery fees, bank suspensions, reward prizes) for evaluation purposes, not verbatim captured real emails.
